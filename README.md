# 🐍 Exercícios de Python - Curso em Vídeo

Este repositório contém todos os desafios e exercícios práticos que realizei durante o curso de **Python 3** do canal [Curso em Vídeo](https://www.youtube.com/cursoemvideo), ministrado pelo professor Gustavo Guanabara.

## 🚀 O que estou aprendendo:
* **Mundo 1:** Fundamentos, tipos primitivos, operadores aritméticos e módulos.
* **Mundo 2:** Estruturas de controle (if, elif, for, while).
* **Mundo 3 (Em andamento):** Estruturas compostas (Tuplas, Listas, Dicionários), Funções e Modularização.

## 📂 Estrutura do Repositório
Os arquivos estão organizados por aulas e temas específicos conforme o cronograma do curso:
- `aula12/`: Estruturas de controle aninhadas.
- `reforço/`: Exercícios extras para fixação de lógica.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.13
* **IDE:** PyCharm Professional
